## Hi — I'm Maheer

এই README-এ সরাসরি CSS দিয়ে রঙ বদলানো কাজ করবে না। এখানে দুইটি কাজ করার উপায় দেয়া হলো:

1. **Badge (shields.io)** — ইমেজ হিসাবে রঙিন টেক্সট দেখাবে (সহজ ও দ্রুত):

    ```md
    ![Hello badge](https://img.shields.io/badge/Hi--Maheer-red)
    ```

    উদাহরণ (রিয়েল টাইম):

    ![Hello badge](https://img.shields.io/badge/Hi--Maheer-red)

2. **Inline SVG image** — আপনি একটি SVG ফাইল রেপোতে যোগ করে তা README-এ ইমেজ হিসেবে ইনক্লুড করতে পারেন, SVG-এ স্টাইল থাকা যায় এবং রঙ দেখায়:

    ```md
    ![colored text](assets/colored_text.svg)
    ```
